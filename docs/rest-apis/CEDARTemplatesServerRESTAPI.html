CEDAR Templates Server REST API

The server supports three types of resources: templates, template elements, and template instances. Templates and template elements are represented using JSON Schema and are annotated using JSON-LD. Template instances are represented using JSON and are also annoted with JSON-LD. No other formats are supported.

Descriptions of these representations are detailed in the "CEDAR Repository Model" document.

Resource Index

These are the resources.

  • /templates [POST, GET, PUT, DELETE]
  • /template_elements [POST, GET, PUT, DELETE]
  • /template_instances [POST, GET, PUT, DELETE]

Resources

/templates/{id}

Templates are represented using JSON Schema. This representation is detailed in the "The CEDAR Repository Model" document.

Methods

Get a template.

Example request URI(s):

  • http://metadatacenter.org/templates/
  • http://metadatacenter.org/templates/basic_study_design

Request query parameters:

parametervaluedescription

id

string

the id of the template

Available response representations:

  • 200 application/json [expand]

    Example

                    "title": "Basic study design",
                    "description": "Basic example of a template to describe a study",
                    "properties": {
                    "@context": {
                    "properties": {
                    "brief_title": {
                    "enum": ["https://schema.org/title"]
                    },
                    "principal_investigator": {

                    ]
                    }
                    },
                    "required": ["brief_title", "principal_investigator"],
                    "additionalProperties": false
                    },
                    "@id": {
                    "type": "string",
                    "format": "uri"
                    },

                    },
                    "brief_title": {
                    "$ref": "http://metadatacenter.org/schemas/brief_title"
                    },
                    "principal_investigator": {
                    "$ref": "http://metadatacenter.org/schemas/principal_investigator"
                    }
                    },
                    "required": ["@context", "@id", "@type","brief_title",

                    ],
                    "additionalProperties": false
                    }

  • 401 If calling user does not have permission to view the template

POST

Create a template element.

Acceptable request representations:

  • 200 application/json [expand]

    Example

    An example of a template element describing a principal investigator.

                        "$schema": "http://json-schema.org/draft-04/schema#",
                        "@id": "http://metadatacenter.org/template_elements/principal_investigator",
                        "title": "Principal Investigator",
                        "description": "The lead researcher for the study",
                        "properties": {
                        "@context": {
                        "properties": {
                        "full_name": {
                        "enum": ["https://schema.org/name"]
                        },
                        "institution": {
                        "enum": ["http://metadatacenter.org/properties/institution"]
                        }
                        },
                        "required": ["full_name", "institution"],
                        "additionalProperties": false
                        },
                        "@type": {
                        "enum": ["http://ncicb.nci.nih.gov/xml/owl/EVS/Thesaurus.owl#C19924"]
                        },
                        "full_name": {
                        "$ref": "http://metadatacenter.org/schemas/full_name"
                        },
                        "institution": {
                        "$ref": "http://metadatacenter.org/schemas/institution"
                        }
                        },
                        "required": ["@context", "@type", "full_name", "institution"],
                        "additionalProperties": false
                        }

Responses:

  • 201 If successfully created.
  • 401 If calling user is not authorized to create a template element
